## ğŸ“¥ Installation & Setup

### **For Users (Recommended)**

1. **Download the latest APK** from the [Releases Page](https://github.com/FarhanZafarr-9/FlexPrime/releases/latest)
2. **Install on your Android device** (Enable "Install from Unknown Sources" if needed)
3. **Launch the app** and login with your university credentials
4. **Explore!** Your data will sync automatically

### **For Developers** (open sourcing soon)

```bash
# Clone the repository
git clone https://github.com/FarhanZafarr-9/FlexPrime.git
cd FlexPrime

# Install dependencies
npm install

# Start the development server
npx expo start

# Run on Android
npx expo run:android

# Run on iOS (macOS only)
npx expo run:ios
```

### **Environment Setup**

Create a `.env` file in the root directory:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
API_BASE_URL=https://flexstudent.nu.edu.pk
```
